Запуск скрипта:

```bash
python <путь до проекта>\main.py --path <путь до файла из которого нужно получить субтитры>
```

пример запуска (из дериктории проекта):
```bash
python .\main.py --path 'C:\Users\Acer\Downloads\DMX - X Gon'' Give It To Ya.mp3'
```

Промежуточный результат ковертации лежит по пути: <br>audio/audio.wav

Результат складывается по пути: <br> result/subtitles.srt
